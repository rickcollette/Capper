We need to create a plan to incorporate ../CapStart as a core foundation of CapperVM.

In the same way people provide lxe recipes to ProxMox, we should be able to provide CapStart Recipes to CapperVM and have it create everything we need.

A good example would be a PiHole or a *arr suite or a Minecraft server.

We also need to be able to upload an OS installation ISO and perform a complete installation into a VM with it.
